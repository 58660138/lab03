การสร้างฟอร์มรับข้อมูล
รูปแบบการกำหนดคุณสมบัติให้แท็ก < form>
< form name="ชื่อฟอร์ม" method="วิธีส่งข้อมูล"
action="ชื่อไฟล์สคริปต์ที่จะเรียกขึ้นมาใช้งาน">

สำหรับวิธีส่งข้อมูลมี 2 รูปแบบคือ POST กับ GET ซึ่งแตกต่างกัน โดยใช้งานกันตามความเหมาะสม
โดยที่ความแตกต่างระหว่าง POST กับ GET คือ

  • POST เป็นการส่งข้อมูลจากฟอร์มไปยังสคริปต์โดยตรง โดยไม่ผ่าน URL ทำให้เราไม่
    สามารถมองเห็น ข้อมูลดังกล่าวได้ด้วยวิธีปกติ แบบนี้จึงเหมาะกับข้อมูลขนาดใหญ่ๆ หรือข้อมูลที่ต้อง-
    การปิดเป็นความลับ เช่นหน้าจอล็อกอิน
  • POST เป็นการส่งข้อมูลจากฟอร์มผ่านทาง URL ของเว็บเพจโดยตรง วิธีนี้ไม่ปลอดภัย-
    เพราะข้อมูลที่ส่งผ่านไปให้สคริปต์นั้น จะถูกแสดงออกมาในรูปแบบของข้อความธรรมดา ใน URL ซึ่งเรา
    จะเห็นในช่อง Address Bar ของบราวเซอร์

    องค์ประกอบต่างๆ ของฟอร์มรับข้อมูล
    การใช้กล่องรับข้อความ (TextField)
    กล่องรับข้อความบรรทัดเดียว (Single line TextField)
    กล่องรับข้อความบรรทัดเดียวจะใช้กรอกข้อความสั้นๆ เช่นชื่อ, นามสกุล, เบอร์โทรศัพท์, หรืออีเมล์
    เป็นต้น
    รูปแบบการเขียนสคริปต์ผ่านแท็ก < input> ในภาษา html โดยกำหนดคุณสมบัติดังนี้
    < input type="text" name="ชื่อกล่องข้อความ"
    size="ความกว้างกล่องรับข้อความ"
    maxlength="จำนวนตัวอักษรสูงสุด">;

    กล่องรับรหัสผ่าน (PasswordField)
    กล่องรับรหัสผ่านจะรับข้อมูลเป็นรหัสผ่านโดยเฉพาะ เนื่องจากผู้ใช้กรอกข้อมูลลงในกล่องรับรหัสผ่านจะปรากฏเป็น
    เครื่องหมายดอกจัน (*) แทนตัวอักษรแต่ละตัวในรหัสผ่าน
    รูปแบบการเขียนสคริปต์ผ่านแท็ก < input> ในภาษา html โดยกำหนดคุณสมบัติดังนี้
    < input type="password" name="ชื่อกล่องข้อความ"
    size="ความกว้างกล่องรับข้อความ"
    maxlength="จำนวนตัวอักษรสูงสุด">;

    กล่องรับข้อความหลายบรรทัด (TextArea)
    กล่องรับข้อความหลายบรรทัดเหมาะสำหรับเป็นกล่องรับข้อความยาวๆ เช่น กล่องรับที่อยู่ หรือกล่องรับความ
    คิดเห็น เป็นต้น
    รูปแบบการเขียนสคริปต์ผ่านแท็ก < textarea> ในภาษา html โดยกำหนดคุณสมบัติดังนี้
    < textarea name="ชื่อกล่องข้อความ" cols="จำนวนของตัวอักษรใน 1 บรรทัด" rows="จำนวนบรรทัด">
    < /textarea>

    การใช้ปุ่มตัวเลิอก
    การสร้างข้อมูลเป็นแบบปุ่มตัวเลือกซึ่งมีอยู่ 2 ประเภทหลักด้วยกันคือ
  • ปุ่มตัวเลือกแบบปุ่มกลม (Radio)
  • ปุ่มตัวเลือกแบบปุ่มเหลี่ยม (Checkbox)

    ปุ่มตัวเลิอก Radio
    ลักษณะของ ปุ่มตัวเลือกแบบ Radio คือ ใน 1 ชุดคำถามผู้ใช้สามารถเลือกคำตอบได้เพียงแค่ 1 คำตอบเท่านั้น เช่นข้อ
    มูลประเภท เพศ, ศาสนา หรือข้อสอบที่เลือกตอบได้เพียงคำตอบเดียวเท่านั้น
    รูปแบบการเขียนสคริปต์ผ่านแท็ก < input> ในภาษา html โดยกำหนดคุณสมบัติดังนี้
    < input type="radio" name="ชื่อของกลุ่มตัวเลือก" value="ค่าของตัวเลือก"/>
    < lable>ข้อความที่ต้องการให้แสดงหน้าปุ่ม< /lable>

    ปุ่มตัวเลิอก CheckBox
    ลักษณะของปุ่ม Checkbox คือสามารถเลือกได้มากกว่า 1 ตัวเลือก เช่น เลือกเกี่ยวกับความชอบ หรือเลือกข้อสอบที่ให้เลือก
    ตอบมากกว่า 1 ข้อ เป็นต้น
    รูปแบบการเขียนสคริปต์ผ่านแท็ก < input> ในภาษา html โดยกำหนดคุณสมบัติดังนี้
    < input type="checkbox" name="ชื่อของกลุ่มตัวเลือก" value="ค่าของตัวเลือก" />
    ข้อความประกอบ

    การใช้กล่องตัวเลือกแบบป็อบอัพเมนู
    เป็นกล่องตัวเลือกแบบแสดงรายการขึ้นมาให้เห็นเพียงรายการเดียว เมื่อคลิกที่ลูกศร เมนูรายการอื่นๆ จึงจะปรากฏขึ้นมาให้
    คุณเลือก
    รูปแบบการเขียนสคริปต์ผ่านแท็ก < select> ในภาษา html โดยกำหนดคุณสมบัติดังนี้
    < select name="ชื่อกล่องตัวเลือก">
    < option value="ค่าของตัวเลือกที่ 1" selected="select">ชื่อตัวเลือกที่ 1< /option>
    < option value="ค่าของตัวเลือกที่ 2">ชื่อตัวเลือกที่ 2< /option>
    < option value="ค่าของตัวเลือกที่ n">ชื่อตัวเลือกที่ n< /option>
    < /select>

    การใช้ปุ่มสั่งงาน
    ปุ่มสั่งงานมี 3 ประเภท ได้แก่ ปุ่มส่งข้อมูล, ปุ่มล้างข้อมูล, ปุ่มสั่งงานแบบรูปภาพ

    ปุ่มส่งข้อมูล (Submit)
    ปุ่มส่งข้อมูล ทำหน้าหน้าที่ส่งข้อมูลที่ผู้ใช้กรอกไปประมวลผลต่อยังสคริปต์ที่ต้องการ
    รูปแบบการเขียนสคริปต์ผ่านแท็ก < input> ในภาษา html โดยกำหนดคุณสมบัติดังนี้
    < input type="submit" name="ชื่อปุ่ม" value="ข้อความหน้าปุ่ม"/>

    ปุ่มล้างข้อมูล (Reset)
    ปุ่มล้างข้อมูล ทำหน้าที่ลบข้อมูลทั้งหมดที่กรอกเข้ามาในฟอร์มขณะนั้นออกไป เหมือนกับการเคลียร์ค่าทั้งหมด
    เพื่อกรอกใหม่นั่นเอง
    รูปแบบการเขียนสคริปต์ผ่านแท็ก < input> ในภาษา html โดยกำหนดคุณสมบัติดังนี้
    < input type="reset" name="ชื่อปุ่ม" value="ข้อความหน้าปุ่ม"/>

    ปุ่มรูปภาพสำหรับส่งข้อมูล (ImageField)
    รูปแบบการเขียนสคริปต์ผ่านแท็ก < input> ในภาษา html โดยกำหนดคุณสมบัติดังนี้
    < input type="image" name="ชื่อปุ่ม" src="ชื่อไฟล์รูปภาพ"/>

    จากนั้นสร้างสคริปต์รับข้อมูลจากฟอร์ม
    เริ่มต้นด้วยการประกาศตัวแปรเพื่อรับค่าจากฟอร์ม เช่น $username=$_POST['username']; username
    ที่อยู่ในวงเล็บ $_POST นั้นคือชื่อของกล่องรับข้อมูลที่อยู่ในฟอร์ม < input name="username" type="text>
    นั่นเอง ชื่อตัวแปรนั้นไม่จำเป็นต้องเหมือนกับชื่อที่ส่งมาจากฟอร์มก็ได้ ตั้งให้เข้าใจว่าเป็นตัวไหนก็พอ
    จากนั้นทำการแสดงผลด้วยคำสั่ง echo
    สคริปต์ register.php
    ข้อมูลของคุณ

    //----------ประกาศตัวรับค่าที่ส่งมาจาก-form-1----------------

    ?>
